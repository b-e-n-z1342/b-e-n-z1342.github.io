Fast-chroot

Современная замена artix-chroot — написана на Go, не зависит от bash, работает мгновенно и без лишних зависимостей. Используется по умолчанию в QuasarLinux.

GitHub QuasarLinux

Зачем это нужно?

Без bash, без проблем

Оригинальный artix-chroot — это bash-скрипт. Fast-chroot написан на Go и не требует интерпретатора, что делает его надёжнее и быстрее.

Минимализм и переносимость

Один статический бинарник — никаких зависимостей. Можно копировать в любую систему и использовать сразу.

Полная совместимость по интерфейсу

Использует тот же синтаксис: fast-chroot /mnt — и вы внутри chroot с правильными точками монтирования (/proc, /sys, /dev, /run).

Как использовать

Установите или скопируйте бинарник, затем выполните:

fast-chroot /путь/к/корню

Вы окажетесь в chroot-окружении с уже смонтированными системными директориями. Выйти — как обычно: exit.

Поддерживает флаги, например:

fast-chroot --help
fast-chroot -h show help
fast-chroot -r do not update resolv.conf
fast-chroot -u string
user[:group] to run as

Почему Go?

Go позволяет собирать полностью статические бинарники. Это означает:

  • Нет зависимостей от glibc, bash или других утилит
  • Работает даже в минимальных live-образах
  • Быстрый запуск и низкое потребление памяти
  • Легко поддерживать и расширять
